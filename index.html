KeyboardMouse
Control your mouse cursor using keyboard. For Windows Vista/7/8/8.1/10 users, it's recommended that you run this program as administrator. If not, it will stop working when your current application has higher privileges, such as Registry Editor.

It uses WH_KEYBOARD_LL hook, so it won't work on Win9x.

temClock v1.1
Amazing Win32 clock application. Super low resource consumption. Works on every Windows version since Win95. Featuring 3 modes (clock, timer and stopwatch) that can work simultaneously. Support setting font and timeout action (like executing program).

As of v1.1, you can enable DwmExtendFrame to get Aero-Glass look.
